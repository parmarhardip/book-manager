A WordPress plugin to manage books using a custom post type and Gutenberg block.

== Description ==

**Book Manager** is a WordPress plugin that allows you to manage books with a custom post type and display them using a Gutenberg block. The plugin includes features to approve or reject book submissions and send email notifications to the authors.

**Features:**
- Custom post type for books
- Book submission form with AJAX handling
- Approval and rejection of books via WP List Table
- Email notifications for book approval and rejection
- Gutenberg block to display books in a customizable layout
- Admin interface to manage book submissions

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/book-manager` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to the 'Books' menu in the WordPress admin to manage your books.

== Frequently Asked Questions ==

= How do I use the Book Manager block? =

Simply add the "Book Manager" block to any post or page using the Gutenberg editor. Configure the settings as needed.

= How can I submit a book? =

A book can be submitted using the shortcode `[add-book]`, which will display a book submission form on any page or post.
